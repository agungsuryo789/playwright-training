# Playwright Training

## Clone the Project

```bash
git clone <your-repo-url>
cd playwright-training
```

## Install Dependencies

```bash
pnpm install
pnpm run install:browsers
```

## Run Tests

```bash
pnpm test
```

## Environment Setup

Copy `.env.example` to `.env` and update as needed.

```bash
cp .env.example .env
```
